# Changelog

## [1.1.0] — 2026-07-15

- `templates.md` в tz-elicit: шаблоны формулировок требований — EARS
  по-русски (всеобщее/событийное/состояние/нежелательное/опциональное),
  user story с критериями Given-When-Then, таблица НФТ с колонкой «как
  измеряем». tz-write ссылается на них в шаге черновика.
- Новый скилл knowledge-map (14-й): воспроизводимая карта корпуса —
  навигатор по доменам wiki (что это / вход / ключевое / читать), синтез
  параллельными scout'ами. После обновления снапшота пересинтезируются
  только домены, задетые `_CHANGES`.
- `--unroll-pre` в convert.py и fix_tables.py: таблицы с многострочными
  JSON/XML-примерами разворачиваются — в ячейке остаётся «см. Пример N
  ниже», код выносится после таблицы фенсами с языком (из class или
  детерминированного сниффа), содержимое байт-в-байт. На реальном корпусе:
  140 таблиц в 109 файлах ушли в чистый GFM, идемпотентно, wikilinks не
  тронуты. Флаг опциональный, потому что меняет вёрстку страницы; без него
  поведение прежнее.
- test/run.sh: 48 самотестов (добавились unroll и провязка templates).

## [1.0.0] — 2026-07-15

Первый публичный релиз / first public release —
github.com/bogdanov-igor/loft.

Ядро документной работы для Claude Code:

- Контракт на 81 строку (~3k токенов always-on): профессия
  системного/бизнес-аналитика и указатели; «ни одного выдуманного факта»;
  правило границы — скрипт в ядре только детерминированный конвертер,
  проверки смысла — указания агентам.
- 13 lazy-скиллов: tz-write, tz-adapt, tz-audit, tz-elicit (банки вопросов
  по 6 типам задач), ingest-confluence, ingest-docs, review-intake (возврат
  заказчика с track changes — правки не теряются), link-check, deliver-pdf
  (цепочка движков typst → weasyprint → wkhtmltopdf → Chrome headless →
  HTML), spec-bootstrap, remember, stage, migrate-specos (карантин машинерии
  предшественника с манифестом отката).
- Конвертер Confluence: детерминированный (pandoc + lxml, без LLM-рерайта),
  собственный писатель таблиц HTML→GFM с логом fallback'ов, имена вложений
  из alias, stale-removal с защитой рукописных файлов, `--space` /
  `--base-url`, отчёт изменений при повторном ingest (`_CHANGES-<дата>.md` +
  `.ingest.json`, sha1-хэши без ложных срабатываний). Обкатан на реальном
  корпусе: 446 страниц, 0 битых из 3645 ссылок, дерево байт-в-байт с
  эталоном.
- 2 сабагента (scout, verifier), 2 хука (leak-guard, update-check: локальные
  источники + GitHub-релизы, кэш 24 ч, тихий), файловая память,
  BACKLOG/QUESTIONS.
- Жизненный цикл: install.sh с самопроверкой и бэкапами; обновление той же
  командой; корпус — граф Foam/Obsidian из коробки (инсталлер сеет
  рекомендацию расширения); 40+ самотестов как релизный гейт сборки;
  golden-фикстуры экспорта Confluence.
- Документация EN+RU внутри архива, Apache-2.0.
